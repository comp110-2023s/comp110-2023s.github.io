1. Create Bear Class

class Bear and __init__

For this part, you are going to create and initialize a class to represent the Bear living by the river. In the file bear.py, do the following:

  1. Define a class with the name Bear. It should have the attribute age which is an integer.
  2. Define an __init__ method that only takes self as an argument and initializes the following variable self.age as an int with the value 0

get_age method

Still working in bear.py, define a method inside the Bear class called get_age that takes no parameters other than self and returns self.age.

2. Create Fish Class

class Fish, __init__, and get_age

Now we are going to create a class to represent the fish in the river. In the fish.py file, create a class called Fish. Like Bear, it should have the attribute age which is an integer. Define __init__ and get_age methods inside this class the same way you defined them in the Bear class.

3. Create River Class

Now that you have some animals, you can create and populate a river!

class River

In rivers.py, create a class called River with the following attributes:

  • day: an int that tells you what day of the river‚Äôs lifecycle you are modeling
  • bears: a list of Bears that stores the river‚Äôs bear population
  • fish: a list of Fish that stores the river‚Äôs fish population

__init__ method

Define an init method of River that: - takes as parameters: self, num_fish: int, num_bears: int - initializes self.fish to contain num_fish many Fish - initializes self.bears to contain num_bears many Bears - initializes self.day to be 0

Test the __init__ method by constructing a river my_river with 2 Bears and 10 Fish.

view_river() method

Now, you are going to define a method inside your River class called view_riverwhich will print the river status out to you. - It takes no parameters other than self and returns nothing. - It should print the following:

~~~ Day x: ~~~
Fish population: y
Bear population: z

Where x is the current day of the river, y is the number of Fish in the river, and z is the number of Bears in the river.

So, for the river you just constructed, my_river.view_river() should print:

~~~ Day 0: ~~~
Fish population: 10
Bear population: 2
